The terminal that renders
Arabic correctly.

Every other Windows terminal breaks connected Arabic — letters disconnect and the order reverses. EasyTer renders every line through HarfBuzz shaping and the Unicode BiDi algorithm, so Arabic (and Persian, Urdu, and Pashto) come out joined, shaped, and in the right order — even inside vim, git, and Claude Code.

A real terminal, not a demo

EasyTer runs a real pseudo-console (pywinpty / ConPTY) on a VT screen emulator, so every interactive program just works — with correct Arabic on top.

🔤

Correct shaping & BiDi

HarfBuzz shaping + Unicode UAX #9 via Qt's QTextLayout — the engine Linux terminals hand-roll.

🪟

Tabs & split panes

Side-by-side and stacked splits, draggable dividers, pane zoom, broadcast input, reopen last tab.

🎨

15+ themes

Dracula, Nord, Tokyo Night, Catppuccin, Gruvbox… full UI theming, opacity, background image.

📝

Built-in editor

Syntax highlighting and line numbers, handling large files — with correct Arabic there too.

🔍

Logical-order search

Ctrl+F searches the logical text, so Arabic matches the way you actually typed it.

🐍

Python plugin API

Keybindings, commands, themes, event hooks, and status segments — scriptable from ~/.easyter.

⚡

Quake-style hotkey

Ctrl+Alt+` summons or hides EasyTer from anywhere — instant drop-down terminal.

🐚

Any shell

Auto-detects PowerShell, cmd, Git Bash, and WSL. Session save/restore, clickable links, OSC 52 clipboard.

🔒

Local-first, no telemetry

Everything stays on your machine. Free and open source under the MIT license.

Get started in one click

No Python, no pip, no PATH setup. Your settings live in ~\.easyter\, so the exe can run from anywhere — Desktop, USB stick, a tools folder.

⬇ Download the latest EasyTer.exe

Prefer to run from source? Python 3.10–3.14:

# clone and run git clone https://github.com/jaqop/EasyTer.git cd EasyTer pip install -r requirements.txt pythonw EasyTer.py
